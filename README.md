# Built By Denys - Personal Portfolio

A modern, animated portfolio website showcasing creative digital experiences and projects. Built with cutting-edge web technologies to deliver a beautiful, responsive, and interactive personal brand.

## Features

- 🎨 **Beautiful Design** - Gradient UI with smooth animations powered by Framer Motion
- 📧 **Contact Form** - Fully functional email system using Resend with validation
- 🎯 **Portfolio Section** - Showcase your projects and work
- ❓ **FAQ Section** - Frequently asked questions about your services
- 📱 **Fully Responsive** - Mobile-first design with Tailwind CSS
- 🌐 **Multilingual Support** - Content in French (expandable to other languages)
- ⚡ **High Performance** - Next.js 16 with Turbopack for blazing-fast development

## Tech Stack

- **Framework**: Next.js 16.2.1
- **UI Library**: React 19
- **Styling**: Tailwind CSS 3.4
- **Animations**: Framer Motion 12.4
- **Forms**: Formik + Yup validation
- **Email**: Resend API
- **Icons**: React Icons
- **Email Templates**: React Email
- **Language**: TypeScript

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── components/        # Reusable React components
├── pages/            # Next.js pages and API routes
├── data/             # Static data (projects, FAQs)
├── lib/              # Utilities (email sending)
├── services/         # Business logic
└── styles/           # Global CSS
```
